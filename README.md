# Pomodoro Timer

- Pomodoro timer allows you to have 25 minutes of focus time followed by 5 minutes of break time.
- Automatic Focus Time - Break Time Cycle
- Start, Pause & Reset Timer.
- Autumn-themed Style Sheet
